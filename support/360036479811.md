<h1 id="h_01K9AQ8Q6Y1SR0BYK2AS7KFJDF"><strong>Adding a Custom Emoji</strong></h1>
<div class="info-box">
<strong><span style="color: #434343;">Note</span></strong><span style="color: #434343;">: If you're not the server owner or don't have the </span><strong><span style="color: #434343;">Create Expressions</span></strong><span style="color: #434343;"> permission enabled, you won't be able to add emojis to that server.</span>
</div>
<p>Every server by default can add up to 50 custom emojis that any active server members can use within the server. If any user is a Nitro subscriber, they’ll be able to use the custom emojis from any server anywhere in Discord where emojis can be used!</p>
<h2 id="h_01K9AQ4AK84N8SSJDKPQSM8HRW">Through Server Settings</h2>
<p>1. To upload custom emojis, choose the server where you would like to add custom emojis.</p>
<p>2. Head into your <strong>Server Settings</strong> by selecting the drop-down menu located next to the server name. </p>
<p>3. Once redirected, choose the <strong>Emoji </strong>tab to and press <strong>Upload Emoji</strong> to upload a custom emoji. </p>
<p class="wysiwyg-text-align-center"><img src="https://support.discord.com/hc/article_attachments/36138548569623" alt="Navigating to Emoji Tab to Upload an Emoji" width="873" height="484"></p>
<h4 id="h_01K9AQ4AK8MG62FEB43E27YGQV" class="wysiwyg-text-align-center"><span style="color: #666666;">Navigating to Emoji Tab to Upload an Emoji</span></h4>
<p><strong>Keep in mind: </strong></p>
<ul>
<li>Emoji names must be at least 2 characters long and can only contain alphanumeric characters and underscores</li>
<li>Emojis must be under 256KB in size</li>
</ul>
<div class="tip-box">
<strong>Tip:</strong> Any user who is an active member of a server is able to use all uploaded custom emoji within that server, and any users who have a Nitro subscription will be able to use the server’s custom emojis anywhere on Discord where emojis can be used.</div>

<div class="warn-box">

<h1 id="h_01K9AQ4AK8EFA3K3NZ75JH1HBJ">Frequently Asked Questions</h1>
<p><strong>Q: How many custom emojis can I upload to my Discord server?</strong></p>
<p>A: Every server can add up to 50 static custom emojis by default. Servers with Nitro or Nitro Basic subscriptions can add an additional 50 animated custom emojis, for a total of 100 custom emojis.</p>
<p><strong>Q: What are the file requirements for uploading custom emojis?</strong></p>
<p>A: Custom emojis must be under 256KB in size and use supported formats like JPEG, PNG, GIF, or WEBP. Emoji names must be at least 2 characters long and can only contain alphanumeric characters and underscores.</p>
<p><strong>Q: Can I use my server's custom emojis on other servers?</strong></p>
<p>A: Custom emojis are only usable on the server where they were uploaded unless you have Discord Nitro or Nitro Basic. With a Nitro subscription, you can use custom emojis from any server anywhere on Discord.</p>
<p><strong>Q: How do I add animated custom emojis to my server?</strong></p>
<p>A: Upload animated emojis (GIF format) the same way you would upload static emojis through Server Settings &gt; Emoji tab. However, you'll need Nitro or Nitro Basic to unlock the additional 50 animated emoji slots for your server.</p>
<p><strong>Q: Can I use Twitch subscriber emotes on Discord?</strong></p>
<p>A: Yes! Sync your Twitch account with Discord, make sure you're subscribed to the streamer's channel, and join their sub-enabled Discord server. You'll then be able to use their Twitch emotes globally on Discord.</p>
<p><strong>Q: Why are my Twitch emotes appearing as static instead of animated?</strong></p>
<p>A: Animated Twitch emotes sync to Discord as static images only. If you want to use animated versions, you'll need to upload them manually as custom emojis to your server.</p>
<p><strong>Q: Who can upload custom emojis to a server?</strong></p>
<p>A: The server owner and anyone with the Manage Emojis or Create Expressions permission can upload custom emojis to a server.</p>
<p><strong>Q: What's the difference between custom emojis and animated custom emojis?</strong></p>
<p>A: Custom emojis are static images, while animated custom emojis are GIF files that play animations. Both count toward your server's emoji limit, but animated emojis require a separate 50-slot allocation that's only available with Nitro or Nitro Basic.</p>
