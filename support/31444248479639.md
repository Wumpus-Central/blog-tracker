Different servers can use the same tag, though the combination of the tag and icon will help distinguish your community.</p><p><strong>Q: What happens to members' profiles if our server loses the Server Tag perk?</strong></p><p>A: If your server loses the Server Tag perk due to insufficient boosts, the tag will no longer be available for members to wear on their profile. Any members who had the tag applied will have it automatically removed from their profiles.</p><p><strong>Q: Can I change our Server Tag after it's been created?</strong></p><p>A: Yes, server owners and admins can change the Server Tag at any time. However, when you update a Server Tag, members will need to manually reapply the new tag to their profiles.</p><p><strong>Q: Can server members customize the color of the Server Tag on their profile?</strong></p><p>A: No, the color of the Server Tag is set by the server owner/admin and remains consistent for all members who use it.</p><p><strong>Q: How many Server Tags can a user display on their profile at once?</strong></p><p>A: Users can display only one Server Tag on their profile at a time, even if they belong to multiple servers with tags.</p><p><strong>Q: Can I restrict which members can use our Server Tag?</strong></p><p>A: Currently, there's no way to restrict which members can use the Server Tag. Any member of your server can choose to display it on their profile.</p><p><strong>Q: Do Server Tags appear in messages or just on profiles?</strong></p><p>A: Server Tags that are applied to a user’s profile will appear next to their name everywhere on Discord. This includes in chat, member lists, and Direct Messages lists.</p><p><strong>Q: Can I include emoji or special characters in a Server Tag?</strong></p><p>A: Server Tags can consist of alphanumerical characters and certain special characters. Certain characters, spaces, and emojis are not supported at this time.</p><p><strong>Q: Can my server have multiple Server Tags?</strong></p><p>A: No, each server can only have one Server Tag at a time. If you want to change your server's tag, you'll need to update the existing one rather than creating additional tags. Remember that when you update your Server Tag, members will need to reapply the new tag to their profiles.</p><p><strong>Q: Will other users be able to click on my Server Tag to join the server?</strong></p><p>A: Yes, selecting a Server Tag will open the <a href="https://support.discord.com/hc/articles/30715364399511-Server-Profile" target="_blank" rel="noopener noreferrer">Server Profile</a> for that server. If applications are enabled, they can apply to join the server from the server profile. </p><p><strong>Q: Can I use current Server Boosts on my server for Server Tags?</strong></p><p>A: If you are a server admin or owner and you have 3 Server Boosts already available for your server to use, then you can enable Server Tags right away. If you don't have enough Server Boosts available, you may need to change your current Server Level or get more Boosts from your community.</p><p><strong>Q: What happens if I edit or change my current Server Tag in my server? </strong></p><p>A: If you update the Server Tag for your server, server members will have to reapply the tag again for the change to take effect.</p><p><strong>Q: Is there a rate limit for how many times I can change my server tag once it is set?</strong></p><p>A: Yes, there is a rate limit for changing your server tag. You can change your server tag after a certain period of time.</p><p><strong>Q: What are “additional perks” and how are they different from Levels?</strong></p><p>A: Additional perks are new features you can unlock directly with Boosts, separate from your server's Boost Level. Currently, there are two available: Enhanced Role Styles (which add gradient and holographic effects to roles) and Server Tags (4-character identifiers members can display on their profiles across Discord). Unlike traditional server levels, these perks can be unlocked individually without reaching specific level thresholds.</p><p><strong>Q: How much do the new Additional Perks cost?</strong></p><p>A: Enhanced Role Styles and Server Tags each cost 3 Boosts. If you want both perks, you'll need 6 Boosts total (3 for each perk).</p><p><strong>Q: Are Additional Perks replacing the current server level system?</strong></p><p>A: No, this supplements the existing level system. All current server level perks (emoji slots, file upload limits, etc.) remain exactly the same. This new system simply gives you additional ways to use your Boosts for targeted perks without affecting your server's level progression.</p><p><strong>Q: Do Additional Perks use separate Boosts from my server level?</strong></p><p>A: Yes, Boosts used for Additional Perks are dedicated specifically to those perks and don't count toward your server's level progression. Similarly, Boosts allocated to your server level can't be used for Additional Perks. You'll need separate Boosts for each - so if you want Enhanced Role Styles, you need 3 Boosts just for that perk, plus whatever Boosts you want to maintain your current server level.</p><p><strong>Q: What happens to my Additional Perks if my server loses Boosts?</strong></p><p>A: Perks require ongoing Boost allocation to remain active. If your server drops below the required number of Boosts for a perk, that perk will be disabled. Enhanced Role Styles will revert to solid colors, and Server Tags will be removed from members' profiles. Server owners/admins can choose which perks to maintain if Boost counts drop.</p><p><strong>Q: Why can't I unlock perks even though I have enough Boosts?</strong></p><p>A: Make sure you're accessing the Server Boost tab (visible only to server owners/admins) and that your Boosts are "available" rather than already allocated to your server level. Only server members with admin permissions can unlock Additional Perks. You need to have enough unallocated Boosts to enable Additional Perks. </p><p><strong>Q: Can I move Boosts between levels and perks after I've allocated them?</strong></p><p>A: Yes, server owners and admins can reallocate Boosts between server levels and Additional Perks through the Server Boosts tab. However, if you move Boosts away from a perk, that perk will be disabled until you allocate enough Boosts back to it.</p><p><strong>Q: Are Additional Perks a one-time purchase, or do they require ongoing Boosts?</strong></p><p>A: Each additional perk requires an ongoing allocation of its required Boosts to stay active, just like levels. If your server drops below the required number of Boosts, the perk will be disabled until the necessary Boosts are reallocated.</p><p><strong>Q: Can I purchase and manage Additional Perks from both desktop and mobile?</strong></p><p><strong>A:</strong> Additional Perks can currently only be purchased and configured on desktop.</p><p> </p>
