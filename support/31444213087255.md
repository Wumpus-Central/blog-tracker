<h1 id="h_01JT6W75AN16FJWVZ7M2EQMGE9">What are Enhanced Role Styles?</h1>
<p class="wysiwyg-text-align-center"><img src="https://support.discord.com/hc/article_attachments/31809877372311" width="236" height="55"></p>
<h4 id="h_01JT6W75ANN6HR68ZVHDMKHBZK" class="wysiwyg-text-align-center"><span style="color: #666666;" data-darkreader-inline-color="">Example of Holographic Enhanced Role Style</span></h4>
<p><strong>Enhanced Role Styles </strong>help to make certain roles stand out more in your server. You can choose between gradient or holographic. The Enhanced Role Styles perk can be added to a server by applying 3 <a href="https://support.discord.com/hc/en-us/articles/360028038352" target="_blank" rel="noopener noreferrer">Server Boosts</a> as an owner or admin of the server. This additional perk is separate from Boost Levels, so your server doesn’t need to reach a specific Boost Level to access it.</p>
<h1 id="h_01JT6W75AN5YEZQ9RPSENYX9VH">How to Use Server Boosts to Unlock Enhanced Role Styles as a Server Owner/Admin</h1>
<p>The server owner or admin can use any available applied Server Boosts to unlock Enhanced Role Styles.</p>
<p>1. With the correct server selected, select <strong>Server Boosts</strong> in the upper-left corner. Only server admins and owners can view this tab.</p>

<p>2. In the upper-right, you’ll see how many Server Boosts you have available and the recent activity of Server Boosts. If you have 3 Server Boosts available, you can enable this perk right away.</p>

<p>3. To apply to <strong>Enhanced Role Styles</strong>, hover over the perk and press <strong>Enable Perk</strong>. A window will appear where you can configure your newly unlocked perk!</p>

<h1 id="h_01JT6W75ANAXRWW3Y6FF9VAPZ6">Frequently Asked Questions</h1>
<p><strong>Q: If I set an Enhanced Role Style for a role, what happens to the role color when my server no longer has the available Server Boosts?</strong></p>
<p>A:If your server loses the Enhanced Role Styles perk due to insufficient boosts, any roles using gradient or holographic styles will automatically revert to solid colors using the primary color from their enhanced style.</p>
<p><strong>Q: Can I use Enhanced Role Styles for all roles in my server?</strong></p>
<p>A: Yes! Once you've unlocked Enhanced Role Styles by applying 3 Server Boosts, you can apply gradient or holographic styles to as many roles as you want.</p>
<p><strong>Q: How do Enhanced Role Styles appear on mobile devices?</strong></p>
<p>A: Enhanced Role Styles are fully supported on all Discord platforms, including mobile. Members will see gradient and holographic roles on iOS and Android just as they appear on desktop/browser.</p>
<p><strong>Q: Can I combine Enhanced Role Styles with other role features like role icons?</strong></p>
<p>A: Absolutely! Enhanced Role Styles work alongside all other role features, including role icons, so you can create truly distinctive roles for your community.</p>
<p><strong>Q: If I switch from gradient to holographic (or vice versa), will I need to spend additional boosts?</strong></p>
<p>A: No. Once Enhanced Role Styles are unlocked for your server, you can freely switch between gradient and holographic styles for any role without using additional boosts.</p>
<p><strong>Q: Do members need to have a specific Discord version to see Enhanced Role Styles?</strong></p>
<p>A: Members will need a reasonably current version of Discord to see Enhanced Role Styles. Anyone using an outdated app version may only see the solid color version of the roles.</p>
<p><strong>Q: How do Enhanced Role Styles appear in dark mode vs. light mode?</strong></p>
<p>A: Enhanced Role Styles are designed to look great in both dark and light mode, with automatic adjustments to ensure visibility and aesthetic appeal in either theme.</p>
<p><strong>Q: How can I disable the animation effect for Enhanced Role Styles?</strong></p>
<p>A: Select the <strong>cogwheel </strong>[<img src="https://support.discord.com/hc/article_attachments/31809886435863" width="20" height="20">] next to your Discord username to access your <strong>User Settings</strong>, select the <strong>Accessibility</strong> tab, and then enable <strong>Enabled Reduced Motion.</strong></p>
<p><strong>Q: Can I add Enhanced Role Styles to a server I'm a member of?</strong></p>
<p>A: If you're a server member who wants to enable Enhanced Role Styles for your server, you can Boost your server to ensure there are enough Boosts available for a perk like this. However, only server admins &amp; owners can enable this perk and configure it for their server.</p>

<p><strong>Q: How can I use the current Server Boosts on my server for Server Tags instead?</strong></p>
<p>A: As a server owner or admin, you can disable the currently applied boosts to a level. To do so, head to the <strong>Server Boosts</strong> tab and then press on <strong>Disable Perk</strong>. You’ll then receive a confirmation window press on <strong>Disable</strong> to confirm. You can use all available Server Boosts to unlock <a href="#h_01JT6W75AN5YEZQ9RPSENYX9VH">additional perks</a>.</p>
<p><strong>Q: What are “additional perks” and how are they different from Levels?</strong></p>
<p>A: Additional perks are new features you can unlock directly with Boosts, separate from your server's Boost Level. Currently, there are two available: Enhanced Role Styles (which add gradient and holographic effects to roles) and Server Tags (4-character identifiers members can display on their profiles across Discord). Unlike traditional server levels, these perks can be unlocked individually without reaching specific level thresholds.</p>
<p><strong>Q: How much do the new additional perks cost?</strong></p>
<p>A: Enhanced Role Styles and Server Tags each cost 3 Boosts. If you want both perks, you'll need 6 Boosts total (3 for each perk).</p>
<p><strong>Q: Are additional perks replacing the current server level system?</strong></p>
<p>A: No, this supplements the existing level system. All current server level perks (emoji slots, file upload limits, etc.) remain exactly the same. This new system simply gives you additional ways to use your Boosts for targeted perks without affecting your server's level progression.</p>
<p><strong>Q: Do additional perks use separate Boosts from my server level?</strong></p>
<p>A: Yes, Boosts used for additional perks are dedicated specifically to those perks and don't count toward your server's level progression. Similarly, Boosts allocated to your server level can't be used for additional perks. You'll need separate Boosts for each - so if you want Enhanced Role Styles, you need 3 Boosts just for that perk, plus whatever Boosts you want to maintain your current server level.</p>
<p><strong>Q: What happens to my additional perks if my server loses Boosts?</strong></p>
<p>A: Perks require ongoing Boost allocation to remain active. If your server drops below the required number of Boosts for a perk, that perk will be disabled. Enhanced Role Styles will revert to solid colors, and Server Tags will be removed from members' profiles. Server owners/admins can choose which perks to maintain if Boost counts drop.</p>
<p><strong>Q: Why can't I unlock perks even though I have enough Boosts?</strong></p>
<p>A: Make sure you're accessing the Server Boost tab (visible only to server owners/admins) and that your Boosts are "available" rather than already allocated to your server level. Only server members with admin permissions can unlock additional perks. You need to have enough unallocated Boosts to enable additional perks. </p>
<p><strong>Q: Can I move Boosts between levels and perks after I've allocated them?</strong></p>
<p>A: Yes, server owners and admins can reallocate Boosts between server levels and additional perks through the Server Boosts tab. However, if you move Boosts away from a perk, that perk will be disabled until you allocate enough Boosts back to it.</p>
<p><strong>Q: Are additional perks a one-time purchase, or do they require ongoing Boosts?</strong></p>
<p>A: Each additional perk requires an ongoing allocation of its required Boosts to stay active, just like levels. If your server drops below the required number of Boosts, the perk will be disabled until the necessary Boosts are reallocated.</p>
<p><strong>Q: Can I purchase and manage additional perks from both desktop and mobile?</strong></p>
<p><strong>A:</strong> additional perks can currently only be purchased and configured on desktop.</p>
