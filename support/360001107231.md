</div><h1 id="h_01KV8Q76EH52VREQZCM6H3705P">What is the Verified Server Program?</h1><p>The Verified Server Program lets official game servers get recognized on Discord. A Verified Server has gone through Discord's verification process, which confirms its connection to a game through Steam.</p><p>This program is currently <strong>exclusively for games</strong> — meaning only a game and its directly associated server can participate. It is not available for other types of communities such as companies, movies, music artists, or general interest servers. To be eligible, your game must be a playable, fully-released game on Steam. Early access counts as released, but games that are not yet available to play – including those with only a "Coming Soon" store page, wishlist page, demo, or playtest—are not eligible. You cannot verify a server before the game it belongs to has launched.</p><div class="warn-box">
<strong>Important:</strong> Previously the Verified Server Program included large companies, films, and musicians. That broader program has been discontinued. The current verified program applies to games only.</div><h1 id="h_01KV8Q76EKBHEP8PNXF8D0SRAB">How Do I Get My Server Verified?</h1><p>Server verification is done through game claiming: Discord's self-serve flow that lets developers prove ownership of their game and verify their server in the process. To claim your game and verify your server, you'll need:</p><ul>
<li data-list-item-id="e8e3d4df858f67713e172538a4af0aa13">A Discord server for your game</li>
<li data-list-item-id="e759fdb40058f009d5fc33f58d31b5934">The game live on Steam, with your Discord invite link added to the Steam store page</li>
<li data-list-item-id="ed393030f06b29d77955673d00ada353d">The game launched and available to download and play</li>
<li data-list-item-id="ec6b83b5ad5ef62c833b0770c9f564ea1">A Team in the Discord Developer Portal, with an Application for your game associated with it</li>
<li data-list-item-id="e21160f5a7166cb4ca14f58cc71dd7feb">The server owner added to that Team, since they'll receive the verification code by email</li>
</ul><p>Once you start the claim, Discord emails a verification code to the server owner, you submit your claim, and our team reviews it. After approval, your server is verified and you gain control over your game's identity on Discord.</p><div class="book-box">
