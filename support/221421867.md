Server owners can always pin messages. Admins can also configure which roles have this permission through <strong>Server Settings &gt; Roles</strong>.</p><p><strong>Q: How many messages can I pin in a channel?</strong></p><p>A: Each channel and DM supports up to 250 pinned messages. Once you reach the limit, you'll need to remove an existing pin before adding a new one.</p><p><strong>Q: Can I pin messages in DMs and group messages?</strong></p><p>A: Yes! Any participant in a DM or group messages can pin messages. There are no permission requirements like there are in servers.</p><p><strong>Q: Does pinning a message notify the person who sent it?</strong></p><p>A: No. When a message is pinned, a system message appears in the channel, but the original sender does not receive a notification.</p><p><strong>Q: Can I pin images, videos, or links?</strong></p><p>A: Yes, you can pin any type of message, including those that contain images, videos, file attachments, or links.</p><p><strong>Q: Will members see when a message is pinned?</strong></p><p>A: Yes. When a message is pinned, a system message appears in the channel to let all members know. This system message itself cannot be pinned.</p><p><strong>Q: Can pinned messages be seen by all members of a channel?</strong></p><p>A: Yes. All members who have access to the channel can view its pinned messages, regardless of when the message was originally sent.</p><p><strong>Q: Why can't I see the Pin Message option?</strong></p><p>A: The Pin Message option only appears if your role has the Manage Messages permission, or if you are the server owner. If you don't see it, contact your server admin to have this permission added to your role.</p>
