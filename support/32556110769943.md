<h2 id="h_01JWV89HC8XC2XSTVX5Z15TQF4">Discord JavaScript Error</h2>
<p>If you receive a JavaScript error message when trying to run Discord, try these steps to resolve this.</p>
<p>1. Delete both of your App Data folders for Discord. </p>
<ul>
<li>Press the Windows key + R and type <code>%appdata%</code>. Select the Discord folder and then delete it.</li>
<li>Press Windows key + R again, and then type <code>%LocalAppData%</code>. Select the Discord folder again, and also delete this one.</li>
</ul>
<p>2. After deleting both App Data folders, reinstall Discord. </p>
