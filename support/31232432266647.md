<h1 id="h_01JR8NDANTXMWANW394DKNADMT">Slash Commands</h1>
<p>Discord's slash commands are a powerful way to interact with <a href="https://support-apps.discord.com/hc/en-us/articles/26577510840087-What-Are-Apps" target="_blank" rel="noopener noreferrer">apps</a> directly in your chat. These commands start with a forward slash (/) and provide an interactive menu of available options.</p>
<p>Slash commands provide these key benefits:</p>
<ul>
<li>Easy-to-use interface with auto-complete suggestions</li>
<li>Built-in command discovery</li>
<li>Standardized command format across all Discord apps</li>
<li>Enhanced security and permissions management</li>
</ul>
<h2 id="h_01JR8NDANTSF2X1QBTGB67HVAE">Using Slash Commands</h2>
<div class="info-box">
<strong>NOTE</strong>: Slash commands only work when an app has been added to a server or your Discord account. Learn how to add apps <a href="https://support-apps.discord.com/hc/en-us/articles/26592957841303-How-to-Discover-and-Add-Apps" target="_blank" rel="noopener noreferrer">here</a>.</div>
<ol>
<li>Type "/" in the message box to see available commands</li>
<li>Browse through the command list or start typing to filter</li>
<li>Select a command and fill in any required parameters</li>
<li>Press Enter to execute the command</li>
</ol>
<p>For a comprehensive guide on using slash commands, including troubleshooting tips and best practices, check out the official<a href="https://support-apps.discord.com/hc/en-us/articles/26501837786775-Slash-Commands-FAQ" target="_blank" rel="noopener noreferrer"> Slash Commands FAQ</a>.</p>
<h1 id="h_01JR8NDANT84ZB41FA0T3TY761">Frequently Asked Questions</h1>
<p><strong>Q: On what platforms can I use keyboard shortcuts? </strong><strong><br></strong>A: Keyboard shortcuts can only be used on the browser and desktop version of Discord. </p>
<p><strong>Q: On what platforms can I use custom keybinds? </strong><strong><br></strong>A: Custom keybinds can only be used on the desktop version of Discord. </p>
<p><strong>Q: What's the quickest way to access Discord's keyboard shortcuts?</strong><strong><br></strong>A: Press <strong>Ctrl + / (Windows) </strong>or<strong> ⌘ + / (Mac)</strong> to open the Keyboard Shortcuts menu for a complete overview of available shortcuts.<br><strong>Q: How do I search for specific messages in Discord?</strong><strong><br></strong>A: You can search within the current channel using <strong>Ctrl + F (Windows) </strong>or<strong> ⌘ + F (Mac)</strong>. To search across all channels, use <strong>Ctrl + Shift + F (Windows)</strong> or <strong>⌘ + Shift + F (Mac)</strong>.<br><strong>Q: Can I customize Discord's keyboard shortcuts?</strong><br>A: Yes, you can create custom keybinds through <strong>User Settings &gt; Keybinds</strong>. This allows you to personalize shortcuts according to your preferences.</p>
<p><strong>Q: How do I quickly switch between servers and channels?</strong><br>A: Use <strong>Ctrl + K (Windows) or ⌘ + K (Mac)</strong> to open the quick switcher. For server navigation, use <strong>Ctrl + Alt + Right/Left (Windows)</strong> or <strong>⌘ + ⌥ + Up/Down (Mac)</strong>.<br><strong>Q: What are slash commands and how do I use them?</strong><br>A: Slash commands are interactive tools accessed by typing "<strong>/</strong>" in the chat. They provide easy access to app and bot commands. Simply type "<strong>/</strong>" to see available commands and follow the auto-complete suggestions.</p>
<p><strong>Q: What slash commands can I use in a Discord voice chat?</strong><strong><br></strong>A: You can use slash commands in Discord voice chats when an app has been added to your server or Discord account. Learn how to add apps <a href="https://support-apps.discord.com/hc/en-us/articles/26592957841303-How-to-Discover-and-Add-Apps" target="_blank" rel="noopener noreferrer">here</a>.<br><strong>Q: How can I format my text in Discord messages?</strong><br>A: Use keyboard shortcuts like <strong>Ctrl/⌘ + B (bold)</strong>, <strong>Ctrl/⌘ + I (italic),</strong> and <strong>Ctrl/⌘ + U (underline)</strong>. You can also use markdown syntax for additional formatting options.<br><strong>Q: Are there accessibility features in Discord?</strong><br>A: Yes, Discord offers comprehensive accessibility navigation using tab, arrow keys, and specific shortcuts for features like screen readers. </p>
<p>These can be found in the <a href="#h_01JR8NDANTVN31N93GJG5EY602">Accessibility Navigation section</a> of the guide. You can find more general information on our Accessibility features <a href="https://discord.com/accessibility">here</a>.<br><strong>Q: How do I manage my audio settings quickly?</strong><br>A: Use <strong>Ctrl + Shift + M (Windows)</strong> or <strong>⌘ + Shift + M (Mac)</strong> to mute/unmute your microphone, and <strong>Ctrl + Shift + D (Windows)</strong> or <strong>⌘ + Shift + D (Mac)</strong> to deafen/undeafen audio.</p>
