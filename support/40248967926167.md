</div><h1 id="h_01KRMMSZEBFSBS2D74H45TYP4V">What is a Spoiler Channel?</h1><p>Spoiler channels provide a space for discussions that are hidden unless members of your server opt to view them. This can include discussions about major plot twists, emotionally heavy topics, or any other sensitive discussions that not all members of your server may want to engage in.</p><p>When pressing on a spoiler channel, members of your server will see a notice that it is a spoiler channel, so they can decide if they want to view that channel’s content. Spoiler channels work similarly to age-restricted channels with one key difference, they do not trigger age verification. Any member of your server can choose to view the contents of a spoiler channel, but until they do it will stay hidden.</p><figure class="wysiwyg-image wysiwyg-image-resized" style="width: 80.07%;"><img style="aspect-ratio: 1838/1306;" src="https://support.discord.com/hc/article_attachments/40509739989143" alt="spoiler-channel-notice-example.png" width="1838" height="1306"></figure><h4 class="wysiwyg-text-align-center" id="h_01KRPPW2SS2Q05J2BP0E2E9HJG">An example of a spoiler channel notice in a Discord server for users to consent to in order to view the contents in a spoiler channel.</h4><p>As with all content on Discord, content in spoiler channels still needs to comply with our <a href="https://discord.com/guidelines" target="_blank" rel="noopener noreferrer">Community Guidelines</a>.</p><h1 id="h_01KRMMZXSDF0NKHEGGY6H38QPZ">How to Set Up a Spoiler Channel</h1><p>Spoiler channels can be set up on desktop or in a web browser. Text channels, voice channels, forum channels, and announcement channels can all be designated as spoiler channels.  </p><p>1. Start by pressing the <strong>Edit Channel</strong> icon [<img src="https://support.discord.com/hc/article_attachments/40509739989655" alt="cogwheel.png" width="23" height="24">] to the right of the channel name in your channel list.</p><p>2. In the channel’s Overview tab, you will see the Content Visibility section. In that section, you’ll see a toggle for <strong>Spoiler Channel</strong>. </p><p>3. Select the <strong>Spoiler Channel</strong> toggle, press <strong>Save Changes</strong> and you’re all set.</p><figure class="wysiwyg-image wysiwyg-image-resized" style="width: 80.16%;"><img style="aspect-ratio: 1378/546;" src="https://support.discord.com/hc/article_attachments/40509739990039" alt="content-visibility-settings-spoiler-channel-options.png" width="1378" height="546"></figure><h4 class="wysiwyg-text-align-center" id="h_01KRPQ1AWNFYHWR3XP6WY9KW8F">In the Edit Channels page for each server channel, you can find the Content Visibility section to enable the spoiler channel setting.</h4><div class="tip-box">
<strong>Tip</strong>: The channel topic will be shown on the spoiler channel notice so that server members can get a sense of what’s inside before opting to view the channel. Including helpful context in your channel topic can help them make an informed decision.</div><h1 id="h_01KRMN3Y2PSSF4ZEMZ6YRYW4N9">Accessing a Spoiler Channel</h1><p>When pressing on a spoiler channel for the first time, you will be shown a notice informing you that you are about to enter a spoiler channel. The notice will include the channel name and channel topic, giving you a chance to continue or to go back. Once you’ve opted to view the channel, you will not see this notice again.</p><h1 id="h_01KRMN5W2HKB9J0QX9FGQXQST0">Frequently Asked Questions</h1><p><strong>Q: When should a spoiler channel be created vs an age-restricted channel?</strong></p><p><strong>A:</strong>  As part of Discord’s <a href="https://discord.com/guidelines" target="_blank" rel="noopener noreferrer">Community Guidelines</a>, server owners must apply an age-restricted label to any channels that contain sexually explicit content. If your channel’s content isn’t adult in nature but you still want users to be able to opt-in to view it, a spoiler channel is the better fit.</p><p><strong>Q: Can an existing age-restricted channel be changed to a spoiler channel and vice versa?</strong></p><p><strong>A:</strong> Yes, to change an existing age-restricted channel to a spoiler channel, you can select the <strong>Spoiler Channel</strong> toggle under <strong>Content Visibility</strong> in your channel settings. However, please keep in mind that sharing sexually explicit content in a channel without an age-restricted label is against our <a href="https://discord.com/guidelines" target="_blank" rel="noopener noreferrer">Community Guidelines</a>.</p><p>To change a spoiler channel to an age-restricted channel, you can select the <strong>Age-Restricted Channel</strong> toggle under <strong>Content Visibility</strong> in your channel settings. Please note that if you change a spoiler channel to an age-restricted channel it will require users to verify their age prior to viewing the channel if they live in regions where <a href="https://support.discord.com/hc/articles/30326565624343" target="_blank" rel="noopener noreferrer">age assurance</a> is live. <br> </p>
