<p>This article provides details on how to report the location of different content types to Discord for alleged copyright and trademark violations. Using the formats detailed in this article allows the Discord team to review your claim efficiently and effectively, so please use them! </p><p><strong>Do not mislead Discord’s support teams.</strong> Do not make false or malicious reports to our Legal Response or other customer support teams, send multiple reports about the same issue, or ask a group of users to report the same content or issue. </ul><h1 id="h_01HER6WT6HJYEV9DAGAPEDAF8G">Where to file a copyright or trademark claim</h1><p>For information on how to file your claim of alleged copyright or trademark infringement, please refer to the Discord Copyright &amp; IP Policy <a href="https://support.discord.com/hc/articles/4410339349655-Discord-s-Copyright-IP-Policy" target="_blank" rel="noopener noreferrer">here</a>.</p><p>In order for our team to review efficiently and effectively, we will need specific formats of information to locate the allegedly infringing material depending on your claim and content type. </p><h3 id="h_01HER70D8DYWD71E2TVQRM1SBZ"><span style="color: #434343; --darkreader-inline-color: #beb8b0;" data-darkreader-inline-color="">Note: Please review the sections before submitting your request. </span></h3><h1 id="h_01HER6WT6HXHBVHR0GT5ZSNJ8R">How to report a user profile or server element</h1><p>Please provide the following information:</p><p>1. The User or Server ID number</p><p>2. Identify the specific allegedly infringing element(s), for example:</p><ul>
<li data-list-item-id="e81707eecb699782f6e6cf4279f17095a">Profile avatar</li>
<li data-list-item-id="e8db06f4ce71b4b1d81c1ceff764b2849">Server icon</li>
<li data-list-item-id="ed914e708b0dec18519e55cdec2558ee5">Server banner</li>
<li data-list-item-id="e03823516990027bf58786a06ff0345fd">Server discovery splash</li>
<li data-list-item-id="eb81961f7e741ef4a321838453f911e74">Server invite splash</li>
</ul><h1 id="h_01HER6WT6H8V7NJ7V0TB7B8HRT">How to report a live stream</h1><p>Please provide the server ID and the channel ID where the alleged infringement is occurring in the following format:</p><p><a href="https://discordapp.com/channels/serverID/channelID/messageID"><span class="wysiwyg-underline">https://discordapp.com/channels/server-ID/channel-ID</span></a>/</p><p>If you are able to provide any evidence you have collected regarding the alleged live stream infringement(s), it may assist us in taking action against the allegedly infringing channel(s) or server(s). </p><h1 id="h_01HER6WT6HP1ATV8HX7J2Y4G4F">How to report a message</h1><p>Please provide the server ID, channel ID and message ID in the following format:</p><p><a href="https://discordapp.com/channels/serverID/channelID/messageID"><span class="wysiwyg-underline">https://discordapp.com/channels/server-ID/channel-ID/message-ID</span></a></p><h1 id="h_01HER6WT6H3GETNPP5T9D5QB1H">How to report a message attachment</h1><p>Please report the direct link to the attachment, which will appear in the below format:</p><p><a href="https://cdn.discordapp.com/attachments/###################/###################"><span class="wysiwyg-underline">https://cdn.discordapp.com/attachments/###################/###################</span></a></p><p>To get the Message Link on iOS, hold down on the Message, then select “Copy Message Link.”</p><p>On Android, hold down on the Message, then select “Copy Message Link.”</p><p>On desktop/browser, “right-click” on the Message, and select “Copy Message Link.”</p><h1 id="h_01HER6WT6HF2X0DNVVTYWRN883">How to report an emoji or sticker</h1><p>Please provide the server ID and the emoji or sticker link. </p><p>To find the emoji or sticker link, right click on the emoji or sticker, select 'Copy Image Address', and paste the resulting URL into your request. The resulting URL should match the below formats:</p><p><a href="https://cdn.discordapp.com/emojis/################.png?v=1"><span class="wysiwyg-underline">https://cdn.discordapp.com/emojis/################.png?v=</span></a>#</p><p><a href="https://media.discordapp.net/stickers/##################.png?size=###"><span class="wysiwyg-underline">https://media.discordapp.net/stickers/##################.png?size=###</span></a></p>
