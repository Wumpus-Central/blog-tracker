</div><h1 id="h_01K6KF55B9F58FPXRQ48BN2ZWS">What is Discord System Helper?</h1><p>Discord System Helper improves Discord's functionality across games and apps by granting Administrator permissions. It helps ensure things like keybinds and other features work reliably while playing games that have elevated permissions.</p><div class="info-box"><span style="color: #434343;"><strong>Note:</strong> Currently, Discord System Helper helps ensure keybinds are working when you are playing a game with elevated permissions. We may add additional functionality in the future to help Discord features work more reliably.</span></div><p>Without Discord System Helper, games with elevated permissions may prevent Discord keybinds from registering, including:</p><ul>
<li>Push-to-Talk hotkeys</li>
<li>Overlay toggle shortcuts and keybinds</li>
<li>Discord-specific gaming shortcuts</li>
</ul><div class="exp-box"><span style="color: #434343;"><strong>Important:</strong> Discord System Helper is currently in experimentation and only available for certain users on Windows desktop clients. </ol><h1 id="h_01K6KF55BGKYG64SZ44ZB0FV6V">Frequently Asked Questions</h1><p><strong>Q: Is Discord System Helper an official Discord tool?</strong></p><p>A: Yes, the Discord System Helper is a verified component developed by Discord. Only specific signed and trusted components can be run through the helper.</p><p><strong>Q: Am I able to remove Discord System Helper once installed?</strong></p><p>A:  Yes. To uninstall Discord System Helper, go to <strong>User Settings &gt; Windows Settings</strong> and select the <strong>Uninstall</strong> option. </p><p class="wysiwyg-text-align-center"><strong><img style="aspect-ratio: 624/416; width: 100%;" class="wysiwyg-image-resized" src="https://support.discord.com/hc/article_attachments/35371304842391" width="624" height="416"></strong></p><h4 class="wysiwyg-text-align-center" id="h_01K6KF55BJWCR4VVE423AS998X"><span style="color: #666666;">Discord System Helper Uninstall Option in Windows Settings</span></h4><p><strong>Q: Will Discord System Helper affect my game performance?</strong></p><p>A: No, Discord System Helper is designed to have minimal impact on system performance and does not interfere with game operations.</p><p><strong>Q: What happens if I don't install Discord System Helper?</strong></p><p>A: Your keybinds may not work in games that run with elevated permissions, but Discord's core functionality will remain unaffected.</p><p><strong>Q: Can I use Discord without System Helper?</strong></p><p>A: Yes, Discord will continue to work normally.
