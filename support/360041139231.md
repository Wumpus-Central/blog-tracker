<h1 id="h_01K9AMB184HCWNVTWAVNYCWVMF">How to Give or Remove Expression Permissions</h1>
<div class="exp-box">
<strong><span style="color: #434343;">Important</span></strong><span style="color: #434343;">: We are currently experimenting with permissions, so the changes to the permissions here are not yet live to everyone.</span>
</div>
<p>As a server owner or admin, you can manage who can add, edit, and remove emojis within your server. </p>
<p>If you would like to allow a role to add custom emojis, you can enable the <strong>Create Expressions </strong>permission. </p>
<p class="wysiwyg-text-align-center"><img src="https://support.discord.com/hc/article_attachments/36138589339799" alt="Create Expressions Toggle" width="471" height="100"></p>
<h4 id="h_01K9AMB184J5VMXTVYE8Y6MM60" class="wysiwyg-text-align-center"><span style="color: #666666;">Create Expressions Toggle</span></h4>
<p>Additionally<strong>,</strong> members will need to have the <strong>Manage Expressions</strong> permission enabled if you  would like for them to edit or remove custom emojis.</p>
<p class="wysiwyg-text-align-center"><img src="https://support.discord.com/hc/article_attachments/36138589340951" alt="Manage Expressions Toggle" width="472" height="100"></p>
<h4 id="h_01K9AMB184J08A5MVYRBSVQBGM" class="wysiwyg-text-align-center"><span style="color: #666666;">Manage Expressions Toggle</span></h4>
<p>1. To get started, head into your <strong>Server Settings</strong> by selecting the drop-down menu located next to the server name. </p>
<p class="wysiwyg-text-align-center"><img src="https://support.discord.com/hc/article_attachments/36138548655127" alt="Server Settings Tab" width="812" height="725"></p>
<h4 id="h_01K9AMB184W8ZNE6X53PV7544Y" class="wysiwyg-text-align-center"><span style="color: #666666;">Server Settings Tab</span></h4>
<p>2. Once directed, choose the <strong>Roles</strong> tab and select the specific role you would like to manage. </p>
<p>3. While in the <strong>Edit Role </strong>page, navigate to <strong>Permissions</strong>.</p>
<p>4. You will now be able to toggle the <strong>Create Expressions</strong> and <strong>Manage Expressions</strong> permissions under <strong>General Server Permissions</strong>. </p>
<p>5. Once the permissions have been set, press <strong>Save Changes </strong>to confirm the changes you have made. </p>
<p class="wysiwyg-text-align-center"><img src="https://support.discord.com/hc/article_attachments/36138548655895" alt="Navigate to Roles in Server Settings to Set Create Expressions and Manage Expressions" width="858" height="636"></p>

<h2 id="h_01K9AMB184W34648QSMTTYWJHQ">Through Server Settings</h2>
<p>In addition to your standard universal emojis, you can easily add custom emojis to your server and sync with your favorite streamers to use their integrated Twitch emotes as well.</p>
<p>Just head into the <strong>Server Settings</strong> and select the <strong>Emoji</strong> tab to upload any custom emojis to share with your entire server. </p>
<p class="wysiwyg-text-align-center"><img src="https://support.discord.com/hc/article_attachments/36138548657943" alt="Navigating to Emoji Tab to Upload an Emoji" width="835" height="463"></p>
<h4 id="h_01K9AMB184KF955VVEW2YWNB2B" class="wysiwyg-text-align-center"><span style="color: #666666;">Navigating to Emoji Tab to Upload an Emoji</span></h4>
<p><strong>Keep in mind: </strong></p>
<ul>
<li>Emoji names must be at least 2 characters long and can only contain alphanumeric characters and underscores.</li>
<li>For optimal emoji resolution, you can upload them in sizes up to <strong><em>128x128 pixels</em></strong>, but they will be resized to <strong><em>32x32 pixels</em></strong>. Additionally, the emoji file size must be smaller than <strong>256KB</strong>.</li>
</ul>
<div class="info-box">
<strong><span style="color: #434343;">Note</span></strong><span style="color: #434343;">: Your custom emojis are only usable on the server you've uploaded them to. However, if you have </span><a href="https://discord.com/nitro"><span class="wysiwyg-underline" style="color: #1155cc;">Discord Nitro</span></a><span style="color: #434343;">, you'll be able to use your custom emojis on every server, group, and DM you're in!</span>
</div>
<p>Without a Nitro subscription, custom emojis can <strong>ONLY</strong> be used in the server they were uploaded on. If you try to enter an emoji from another server, only the emoji text code will appear. </p>
<p class="wysiwyg-text-align-center"><img src="https://support.discord.com/hc/article_attachments/36138548658583" alt="Emoji Picker Notification for Nitro Subscription Needed" width="736" height="772"></p>

<div class="info-box">
<strong><span style="color: #434343;">Note</span></strong><span style="color: #434343;">: If you've subscribed to a few partnered Twitch streamers who've enabled their own custom emojis from their stream channel, you can </span><a href="https://support.discord.com/hc/en-us/articles/212112068-Twitch-Integration-FAQ"><span class="wysiwyg-underline" style="color: #1155cc;">sync your Twitch account to Discord</span></a><span style="color: #434343;"> and then use their custom emojis in any server you join!</span>

<h1 id="h_01K9AMB184VPH3PMA6W7ESBH5P">Frequently Asked Questions</h1>
<p><strong>Q: Can I use custom emojis from other servers without Discord Nitro?</strong></p>
<p>A: No, custom emojis can only be used in the server where they were uploaded unless you have Discord Nitro or Nitro Basic. With Nitro, you can use custom emojis from any server across Discord.</p>
<p><strong>Q: How many custom emojis can I add to my server?</strong></p>
<p>A: Every server can add up to 50 custom emojis by default. Servers with Nitro or Nitro Basic subscriptions can add an additional 50 animated custom emojis.</p>
<p><strong>Q: What file formats are supported for custom emojis?</strong></p>
<p>A: You can upload custom emojis in JPEG, PNG, GIF, or WEBP format. The file must be smaller than 256KB.</p>
<p><strong>Q: What are wumboji and how do I use them?</strong></p>
<p>A: Wumboji are enlarged emojis that appear when you send a message containing only emojis with no other text. You can use up to 27 emojis before they return to regular size. Note that wumboji don't appear in Compact mode.</p>
<p><strong>Q: Can I use Twitch emotes on Discord?</strong></p>
<p>A: Yes! If you're subscribed to Twitch streamers who have enabled their custom emojis, you can sync your Twitch account to Discord and use their custom emojis globally across any server.</p>
<p><strong>Q: What permissions do I need to add custom emojis to a server?</strong></p>
<p>A: You need to be the server owner or have the <strong>Create Expressions</strong> permission enabled for your role to add custom emojis to a server.</p>
<p><strong>Q: Why can't I see custom emojis from a specific server in my Emoji Picker?</strong></p>
<p>A: If a server isn't showing in your emoji list, the server admin has disabled the use of custom emojis from other servers in their server settings.</p>
