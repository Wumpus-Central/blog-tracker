<h2 id="h_01K2HNH9NMDCTYASKKV8NPJQW0">Introduction</h2>
<p>We are transitioning from the Legacy Discord Game SDK to the new <a href="https://discord.com/developers/docs/discord-social-sdk/overview" target="_blank" rel="noopener noreferrer">Discord Social SDK</a>! Read on to learn the key differences between these SDKs and why you should migrate to the Discord Social SDK.</p>
<h2 id="h_01K2HNH9NMPCQ69D01T26P4Q22">What is the Discord Game SDK?</h2>
<p>The Discord Game SDK was our original development kit focused on gaming-specific features. Since its shift to Legacy status, developers have primarily used this SDK for its rich presence features. While the Game SDK focused on basic game integration, the Discord Social SDK emphasizes social connectivity and player engagement.</p>
<h2 id="h_01K2HNH9NM0ES5K2HTBTJ2NDRJ">What is the Discord Social SDK?</h2>
<h3 id="h_01K2HNQ523ZNQ0E113P5R0Y80V">Core Features</h3>
<p>The Discord Social SDK is a cross-platform development kit that offers the following core features to all game developers:</p>
<ul>
<li>
<a href="https://discord.com/developers/docs/discord-social-sdk/development-guides/creating-a-unified-friends-list" target="_blank" rel="noopener noreferrer"><strong>Unified Friends List</strong></a> enables players to maintain their social connections across their games and gaming sessions.</li>
<li>
<a href="https://discord.com/developers/docs/rich-presence/using-with-the-discord-social-sdk" target="_blank" rel="noopener noreferrer"><strong>Rich Presence</strong></a> will get your players spreading the word on Discord about your game simply by playing it!</li>
<li>
<a href="https://discord.com/developers/docs/discord-social-sdk/development-guides/managing-game-invites" target="_blank" rel="noopener noreferrer"><strong>Deeplink Game Invites</strong></a> will allow players to invite players so that they can easily join game sessions with just one click.</li>
<li>
<strong><a href="https://discord.com/developers/docs/discord-social-sdk/development-guides/account-linking-with-discord" target="_blank" rel="noopener noreferrer">Account Linking</a></strong> enables Discord-based sign-in and authorization.</li>
</ul>
<h3 id="h_01JP334YAWQ14ESKK4A19FJ2T3">Communication features</h3>
<p id="h_01K2HNN362KQ9KZ6XYGV5WJPMK"><strong>⚠️ The following communication features are available for development and testing, but their usage is capped with a rate limit. Please read our <a href="https://discord.com/developers/docs/discord-social-sdk/core-concepts/communication-features#rate-limits" target="_blank" rel="noopener noreferrer">documentation on rate limits</a> to learn more.</strong><!-- notionvc: 54a1de3c-d4da-468c-9025-da6de11392a9 --></p>
<ul>
<li>
<strong><a href="https://discord.com/developers/docs/discord-social-sdk/development-guides/sending-direct-messages" target="_blank" rel="noopener noreferrer">Cross-platform messaging</a></strong> allows users to message each other in-game and on Discord.</li>
<li>
<strong><a href="https://discord.com/developers/docs/discord-social-sdk/development-guides/managing-voice-chat" target="_blank" rel="noopener noreferrer">Voice chat integration</a></strong> delivers excellent audio quality and lets you create individual user voice streams.</li>
<li>
<strong><a href="https://discord.com/developers/docs/discord-social-sdk/development-guides/linked-channels" target="_blank" rel="noopener noreferrer">Linked Channels</a></strong> provide persistent group chats that allow conversations to flow seamlessly before, during, and after gameplay—both in-game and out.</li>
</ul>

<h1 id="h_01JP332YTM4VA442P8WE6ZQN53">Why should I migrate my game to the Discord Social SDK?</h1>
<p>We suggest migrating from the Legacy Game SDK as it is no longer being maintained. If you would like to migrate over to the Discord Social SDK for your rich presence features and to experience the other <a href="https://support-dev.discord.com/hc/en-us/articles/30127085446039" target="_blank" rel="noopener noreferrer">social features</a> that the SDK offers, check out this <a href="https://discord.com/developers/docs/discord-social-sdk/getting-started" target="_blank" rel="noopener noreferrer">getting started guide</a> to begin!</p>
