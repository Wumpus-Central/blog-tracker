<p>Every Application, Team, and Server in Discord has a unique multi-digit ID. These IDs help our developer support team assist you more effectively. Let's learn how to find them!<!-- notionvc: aab563c6-8631-4698-b5eb-683d1c1429a3 --></p>
<h2 id="h_01JSCWWFKDTZFNA50JM299VP37">Enable Developer Mode</h2>
<p>Head into <strong>User Settings &gt; Advanced</strong> &amp; enable <strong>Developer Mode.</strong></p>
<p><img src="https://support-dev.discord.com/hc/article_attachments/31564919139479" alt="Screenshot_2025-04-21_at_12.26.44_PM.png"></p>
<h2 id="h_01JSCWWFKDRFAFMG2AN08Z13PK">Application ID</h2>
<p>The Application ID, also known as the <strong>Client ID</strong>, can be found in the Developer Portal in <strong>General Information</strong>. </p>
<p><img src="https://support-dev.discord.com/hc/article_attachments/31564919143575" alt="Screenshot 2025-04-21 at 2.31.38 PM.png"></p>
<p>You also can find the Application ID by <strong>right-clicking</strong> on the bot user anywhere that it appears in the Discord Client.</p>
<h2 id="h_01JSCWWFKD6W7JX4M2YGBKADW1">Team ID</h2>
<p>To get the <strong>Team ID</strong> you'll want to head into <strong>Developer Portal &gt; Teams</strong> tab &amp; go to the <strong>URL</strong>. The Team ID will be located here. </p>
<h2 id="h_01JSCWWFKD05WPWBP9SW8HMQYN"><img src="https://support-dev.discord.com/hc/article_attachments/31564934797719" alt="Screenshot 2025-04-21 at 2.47.40 PM.png"></h2>
<h2 id="01JSCYF47R6ZVQ0FDNCFKAMESV">Server ID / Guild ID</h2>
<p>Right-click the server icon &amp; select <strong>Copy ID</strong>. Double check if you have Developer Mode enabled if you're not seeing a Copy ID option. </p>
