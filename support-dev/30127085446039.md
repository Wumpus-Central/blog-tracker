<h2 id="h_01K2HMSEQ0EMJYEQT3ED3RMGRR">Introduction</h2>
<p>Supercharge your game's social experience with the new <a href="https://discord.com/developers/docs/discord-social-sdk/overview">Discord Social SDK</a>! This powerful toolkit allows game developers to power their game's social features, creating a more engaging and connected player experience.</p>
<h2 id="h_01K2HMSEQ0CWHZHAVQAH4M6D9Y">Why should I use the Discord Social SDK in my game?</h2>
<h3 id="h_01K2HMSEQ033R3RW2TRHGCSN4P">Seamless Integration</h3>
<p><img src="https://support-dev.discord.com/hc/article_attachments/30699524966167" alt="UFL_GIF.gif"></p>
<ul>
<li>
<p><strong>Branding integration</strong> allows you to use your own game branding with the SDK for a consistent player experience.</p>
</li>
<li>
<p><strong><a href="https://discord.com/developers/docs/discord-social-sdk/development-guides/creating-a-unified-friends-list" target="_blank" rel="noopener noreferrer">Unified Friends List</a></strong> enables players to maintain their social connections across their games and gaming sessions.</p>
</li>
</ul>
<h3 id="h_01K2HMSEQ0NBWFCJBYE2TBPQBD">Improve Player Growth</h3>
<p><img src="https://support-dev.discord.com/hc/article_attachments/30699524967063" alt="Rich Presence_GIF.gif"></p>
<ul>
<li>
<strong><a href="https://discord.com/developers/docs/rich-presence/using-with-the-discord-social-sdk" target="_blank" rel="noopener noreferrer">Rich Presence</a></strong> will get your players spreading the word on Discord about your game simply by playing it!</li>
<li>
<strong><a href="https://discord.com/developers/docs/discord-social-sdk/development-guides/managing-game-invites" target="_blank" rel="noopener noreferrer">Deeplink Game Invites</a></strong> will allow players to invite new players so that they can discover and join game sessions easily.</li>
<li>
<strong><a href="https://discord.com/developers/docs/discord-social-sdk/development-guides/account-linking-with-discord" target="_blank" rel="noopener noreferrer">Account Linking</a></strong> enables Discord-based sign-in and authorization.</li>
</ul>
<h3 id="h_01K2HMSEQ03R68SBWFAGBQN1FJ">Communication Features</h3>
<p><img src="https://support-dev.discord.com/hc/article_attachments/30699524968471" alt="Cross Platform Chat_GIF.gif"></p>
<p id="h_01K2HMW5ZJY2NV96MS44DBC56R"><strong>⚠️ The following communication features are available for development and testing, but their usage is capped with a rate limit. Please read our <a href="https://discord.com/developers/docs/discord-social-sdk/core-concepts/communication-features#rate-limits" target="_blank" rel="noopener noreferrer">documentation on rate limits</a> to learn more.</strong></p>
<ul>
<li>
<strong><a href="https://discord.com/developers/docs/discord-social-sdk/development-guides/sending-direct-messages" target="_blank" rel="noopener noreferrer">Cross-platform messaging</a></strong> allows users to message each other in-game and on Discord.</li>
<li>
<strong><a href="https://discord.com/developers/docs/discord-social-sdk/development-guides/managing-voice-chat" target="_blank" rel="noopener noreferrer">Voice chat integration</a></strong> delivers excellent audio quality and lets you create individual user voice streams.</li>
<li>
<strong><a href="https://discord.com/developers/docs/discord-social-sdk/development-guides/linked-channels" target="_blank" rel="noopener noreferrer">Linked Channels</a></strong> provide persistent group chats that allow conversations to flow seamlessly before, during, and after gameplay—both in-game and out.</li>
</ul>
<p>For more information on how to access these features, please see <a href="https://discord.com/developers/docs/discord-social-sdk/core-concepts/communication-features" target="_blank" rel="noopener noreferrer">Core Concepts: Communication Features</a>.</p>
<h2 id="h_01K2HMSEQ0EJ5FRM0EB3TCD1BY">What Platforms are supported?</h2>
<p>The Discord Social SDK is available for these platforms:</p>
<h3 id="h_01K2HMSEQ0R6ARRZRMK2M0T7NE">Desktop</h3>
<ul>
<li>Standalone C++ (Windows and macOS)</li>
<li>Unreal engine (Windows Only)</li>
<li>Unity engine (Windows and macOS)</li>
</ul>
<h3 id="h_01K2HMSEQ0HSZE516R751CBRKK">Console</h3>
<p id="h_01K2HN06JTYEPAY2TQ6Z0JPMEH"><strong>⚠️ To use the Discord Social SDK in your console games, you will need to request middleware approval and be an approved developer for the target console. Check out <a href="https://support-dev.discord.com/hc/en-us/articles/30209074764183" target="_blank" rel="noopener noreferrer">this article</a> to learn more.</strong></p>
<p><strong>Xbox Series X|S</strong></p>
<ul>
<li>Standalone C++</li>
<li>Unreal Engine</li>
<li>Unity Engine</li>
</ul>
<p><strong>PlayStation™ 5</strong></p>
<ul>
<li>Standalone C++</li>
<li>Unreal Engine</li>
<li>Unity Engine</li>
</ul>
<h3 id="h_01K2HMSEQ0ATSXV8XQ8K3P0PSB">Android/iOS (full mobile support coming soon)</h3>
<ul>
<li>Standalone C++</li>
<li>Unity engine</li>
</ul>
<h2 id="h_01K2HMSEQ02HJP2PGVF18CRJMG">How Do I Get Started?</h2>
<p>Ready to integrate the Discord Social SDK into your game? Visit our <a href="https://discord.com/developers/docs/discord-social-sdk/overview" target="_blank" rel="noopener noreferrer">Developer Portal</a> to access comprehensive documentation, setup guide, and plugins for popular game engines.</p>
<p>Follow these quick steps to get started:</p>
<ol>
<li>Log in to your <a href="https://discord.com/developers" target="_blank" rel="noopener noreferrer">Discord Developer</a> account</li>
<li>Create a Discord Application to use with the SDK</li>
<li>Follow our <a href="https://discord.com/developers/docs/discord-social-sdk/getting-started" target="_blank" rel="noopener noreferrer">Getting Started Guide</a> in the documentation and review our <a href="https://support-dev.discord.com/hc/articles/30225844245271/" target="_blank" rel="noopener noreferrer">Social SDK Terms</a>
</li>
</ol>
<p>Need more help? Join our <a href="https://discord.com/invite/discord-developers" target="_blank" rel="noopener noreferrer">Discord Developers server</a> to connect with other developers in the <code>#social-sdk-dev-help</code> channel or reach out to our support team <a href="https://support.discord.com/hc/en-us/requests/new?ticket_form_id=12094720423319&amp;tf_12094679025047=field_value_developer_offering_social_sdk">here</a>!</p>
